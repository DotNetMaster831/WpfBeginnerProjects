# 📘 WPF Beginner Projects – Complete Foundation Series

## 🚀 Overview

This repository contains a structured collection of **Beginner-Level WPF projects** built using modern .NET development practices.

The goal of this repository is not just to build small applications, but to:

- Master WPF fundamentals
- Properly implement MVVM architecture
- Apply SOLID principles
- Follow clean coding standards
- Introduce design patterns
- Implement logging and exception handling
- Structure projects professionally
- Prepare for real-world desktop development

This repository is ideal for:

- Beginners starting with WPF
- ASP.NET developers transitioning to Desktop
- Students preparing for interviews
- Developers who want structured WPF learning

---

# 🏗 Architecture Used in All Projects

Every project in this repository follows:

- ✔ MVVM (Model–View–ViewModel)
- ✔ Separation of Concerns
- ✔ Dependency Injection (basic level)
- ✔ SOLID Principles (SRP & DIP focus)
- ✔ Clean Folder Structure
- ✔ Logging with Serilog
- ✔ Global Exception Handling
- ✔ Validation (IDataErrorInfo / ValidationRules)
- ✔ Commands (ICommand / RelayCommand)
- ✔ ObservableCollection & Data Binding
- ✔ Resource Dictionaries for Styling
- ✔ Basic Animations (Storyboard / Triggers)
- ✔ Unit Testing for ViewModels (NUnit/Moq)
- ✔ MSIX / ClickOnce Packaging
